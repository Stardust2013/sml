◆ computeModulo()

template<typename Real >
Real sml::computeModulo ( const Real  dividend,
const Real  divisor 
)

Computes the remainder of division of one real number by another. The remainder is in the range [ 0, divisor ).

The modulo computation is based on the mathematical definition of congruence, which is different from the implementation of std::fmod() in the cmath standard library.

For a description of congruence see: http://mathworld.wolfram.com/Congruence.html.
